# Shopping-Cart-using-Dictionaries-and-Lists

This is a beginner Python program that shows a small food menu, lets the user choose items, and calculates the total bill.

## How to Run

1. Install Python 3
2. Save the code in a file called `menu.py`
3. Open a terminal in the folder
4. Run:

python menu.py

## What the Program Does

- Displays a menu with prices
- Lets the user select items
- Press `q` to stop selecting
- Shows the total bill at the end

## Menu Items

Popcorn - $5  
Soda - $2  
Chips - $1  
Icecream - $3  

## Example

User selects items and the program prints the total price.
